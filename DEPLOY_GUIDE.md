# サイト公開ガイド

完成した「AI Studio g」のウェブサイトをインターネット上に公開する方法をまとめました。
無料で手軽に公開できる **GitHub Pages** と **Google サイト** の2つの方法について手順を解説します。

## おすすめの公開方法

今回のサイトはHTML/CSS/JSで構成された本格的なWebサイトですので、**GitHub Pages** での公開が最も適しています。GoogleサイトはHTMLの直接記述に制限があり、デザインや機能を完全に再現できない可能性があります。

---

## 方法1: GitHub Pages で公開する（推奨）

エンジニアにも人気のある、GitHubの無料ホスティング機能を使う方法です。現在のファイルをそのままアップロードして公開できます。

### 手順

1. **GitHubアカウントの作成**
    * [GitHub公式サイト](https://github.com/)にアクセスし、アカウントを作成します（無料）。

2. **新しいリポジトリの作成**
    * GitHubにログインし、右上の「+」アイコンから「New repository」を選択します。
    * Repository name（例: `my-ai-web`）を入力し、「Public」を選択して「Create repository」をクリックします。

3. **ファイルのアップロード**
    * 作成されたリポジトリ画面で「uploading an existing file」というリンクをクリックします。
    * パソコン上の `antiG103` フォルダ内にある**全てのファイルとフォルダ**（`asset`フォルダ含む）をドラッグ＆ドロップします。
    * 「Commit changes」をクリックして保存します。

4. **公開設定**
    * リポジトリの「Settings」タブを開きます。
    * 左側のメニューから「Pages」を選択します。
    * Branchを「main」または「master」に設定し、「Save」をクリックします。
    * 数分待つと、画面上部に公開URL（例: `https://username.github.io/my-ai-web/`）が表示されます。

---

## 方法2: Google サイト で公開する

Googleのサービスを使って公開する方法です。ただし、作成したHTMLファイルをそのままアップロードすることはできず、Googleサイトの仕様に合わせて「埋め込み」を行う必要があります。

※**注意**: 今回作成したような複雑なCSSアニメーションや複数ページの構成は、Googleサイトでは正しく動作しないか、再現が非常に難しい場合があります。

### 手順（HTMLを埋め込む場合）

1. **Googleサイトの作成**
    * [Googleサイト](https://sites.google.com/)にアクセスし、「空白」の新しいサイトを作成します。

2. **コードの埋め込み**
    * 右側のメニューから「埋め込む」>「埋め込みコード」を選択します。
    * `index.html` の中身（テキストエディタで開いて全コピー）を貼り付けます。
    * **重要**: 画像ファイルやCSS、JSファイルはGoogleサイトに直接アップロードしてもリンクされないため、画像はGoogleドライブ等に置いてURLを指定し直すか、全てを1つのHTMLファイルにまとめる（インライン化）などの高度な修正が必要です。

### 結論

現在の「AI Studio g」のように、画像、CSS、JSが分かれている本格的な構成の場合は、**GitHub Pages（方法1）** が圧倒的に簡単で、デザインもそのまま綺麗に表示されます。

## 公開前の最終チェックリスト

* [ ] **画像パス**: すべての画像が表示されているか？（GitHub PagesならそのままでOK）
* [ ] **リンク**: 全ページのメニューやボタンのリンク切れがないか？
* [ ] **スマホ表示**: スマートフォンで見ても崩れていないか？
* [ ] **SEO情報**: タイトルや説明文は各ページ適切か？（設定済み）

ご不明な点があれば、いつでもご相談ください。
