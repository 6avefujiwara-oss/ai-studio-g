# AIWeb 開発計画書

## 1. プロジェクト概要

AIを活用し、従来よりも「速く」「安く」「高品質」なウェブサイト制作サービス「AIWeb」の公式サイトおよびサービス基盤を構築します。
「未来感」「スピード」「手軽さ」をキーワードに、ユーザーが直感的にサービスのメリットを理解できるデザインを目指します。

## 2. 目的

* **サービスの認知拡大**: AIによるWeb制作という新しい価値観の提示。
* **リード獲得**: 興味を持ったユーザーからの問い合わせ、申し込みへの誘導。
* **信頼性の確保**: プロセスと実績の可視化による安心感の醸成。

## 3. ターゲット層

* 初期費用を抑えたいスタートアップ・個人事業主
* 至急でLPやコーポレートサイトが必要な中小企業
* 最新技術に関心が高い層

## 4. 実装機能一覧 (ロードマップ)

### Phase 1: ランディングページ (MVP) - **[現在進行中]**

ユーザーが最初に訪れる「顔」となるページの構築。

* [x] **ヒーローセクション**: インパクトのあるFV、キャッチコピー。
* [x] **特徴 (Features)**: スピード、コスト、品質の訴求。
* [x] **制作フロー (Process)**: 3ステップで完結する手軽さの提示。
* [x] **料金プラン (Pricing)**: 明確な3つのプラン提示。
* [x] **制作実績 (Portfolio)**: AIによる生成事例のギャラリースライダー。
* [x] **よくある質問 (FAQ)**: 契約や修正に関するQ&A。
* [x] **お問い合わせセクション**: フォームの実装。

### Phase 2: 信頼性とSEOの強化

* [x] **会社概要/運営者情報**: 特商法表記など。
* [x] **プライバシーポリシー**: 個人情報の取り扱いについて。
* [x] **個別ページ化**: LPから各詳細ページへの分割（必要に応じて）。
* [x] **ブログ/ニュース**: SEO流入を狙ったコンテンツ発信基盤。

### Phase 3: システム化 & 自動化 (Future)

* [x] **AI見積もりシミュレーター**: 条件を選ぶだけで概算を出力。
* [x] **デザイン生成デモ**: キーワード入力で簡易プレビューを表示。
* [x] **マイページ**: 制作進捗の確認画面。

## 5. デザインコンセプト

* **テーマ**: Futuristic & Transparent (近未来と透明感)
* **スタイル**: Glassmorphism (すりガラス効果)、ネオングラデーション
* **カラーパレット**:
  * Base: Deep Navy (`#050510`)
  * Primary: Cyan (`#00f2ff`)
  * Secondary: Purple (`#bd00ff`)
  * Text: White / Light Gray

## 6. 技術スタック

* **HTML5**: セマンティックなマークアップ
* **CSS3**: 変数(CSS Variables)を活用した管理、アニメーション
* **JavaScript**: 軽量なDOM操作、スクロールエフェクト (ライブラリ非依存)
